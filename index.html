Choosing a Programming Language

By Kolman Soifer

Choosing languages to begin your programming journey can be difficult. Here is a tool to help you decide. Good luck!

Some languages you may have heard of include:

  • JavaScript
  • Python
  • Hoon
  • Rust
  • C#
  • Go

Regardless of which languages you choose to focus on, learning any of the above will help you on your journey.
